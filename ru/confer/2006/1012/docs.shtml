Министерство здравоохранения Хабаровского края

ПРИКАЗ

11 сентября 2006 г.г. Хабаровск№ 282

О проведении III Дальневосточной конференции
по антимикробной терапии с международным участием


В соответствии с планом работы министерства здравоохранения Хабаровского края на 2006 год ПРИКАЗЫВАЮ:

  1. Провести 12-13 октября 2006 года в г. Хабаровске III Дальневосточную конференцию с международным участием по антимикробной терапии.
  2. Утвердить прилагаемые:
    • состав организационного комитета по подготовке и проведению III Дальневосточной конференции с международным участием по антимикробной терапии;
    • программу III Дальневосточной конференции с международным участием по антимикроб­ной терапии;
    • состав участников III Дальневосточной конференции с международным участием по анти­микробной терапии.
  3. Определить место для проведения III Дальневосточной конференции с международным участием по антимикробной терапии — лекционный зал ГОУДПО «Институт повышения квалификации специалистов здравоохранения» министерства здравоохранения Хабаровского края (г. Хабаровск, ул. Краснодарская, 9).
  4. Руководителям органов управления здравоохранением городских округов и муниципальных районов, государственных учреждений здравоохранения края:

    4.1. Командировать для участия в работе III Дальневосточной конференции с международным участием по антимикробной терапии врачей всех специальностей согласно приложению 3 к настоящему приказу.

    4.2. До 25 сентября 2006 года представить в министерство здравоохранения Хабаровского края персональный список врачей (фамилия, имя, отчество, занимаемая должность), направляемых для участия в работе III Дальневосточной конференции с международным участием по антимикробной терапии.

    4.3. Оплату командировочных расходов произвести по месту работы командируемых в установ­ленном порядке.

  5. Контроль за выполнением настоящего приказа возложить на первого заместителя министра здравоохранения Хабаровского края Балабкина И.Б.
И.о. министраА.Я. Деркач
